jQuery.Magnet

A plugin to drop elements in magnetized zones

Download this project as a .zip file Download this project as a tar.gz file
INNER
Posicionará los elementos con respecto a los bordes a partir de un offset dado.
Se establece una franja con offset desde el centro, tirando una linea vertical a partir del elemento dado, para poder pegar horizontalmente los elementos que se suelten dentro de ella.
Se establece una franja con offset desde el centro, tirando una linea horizontal a partir del elemento dado, para poder pegar verticalmente los elementos que se suelten dentro de ella.
Se centrará los elementos que se suelten en el centro del área magnetizada.
